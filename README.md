# Apartment Finder

Browse for available units at [BJB Properties](https://bjbproperties.com/)! This dashboard scrapes their website once per day for the most up-to-date listings. I am NOT affiliated with BJB Properties, but I have been their tenant for four years and counting. I built this tool to circumvent BJB's website, which forces you to individually click each property to view available units. The search functionality is not any better, either. To prospective tenants, I hope you find this alternative useful!
